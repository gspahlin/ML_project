Using the Machine learning tools to predict diseases based on a set of disease symptoms

A disease dataset was downloaded from https://www.kaggle.com/kumar9249/disease-prediction. This dataset consists of 42 diseases, with 133 associated symptoms. This page describes the results of Decision Tree and Random Forest models used to predict specific diseases based on lists of symptoms.

A summary of the diseases included in this dataset, along with numbers of cases for each

Decision tree

Python was used to develop decision tree models to predict diseases based on disease symptoms. The datset was split into training and testing sets, then decsion trees were generated based on the training set. For this particular example The data was split 75/25 training/test. A figure showing the actual decision tree as visualized using Fraphviz is shown at the bottom of the page. The accuracy for this model was training: 1.0 test: 0.991 A Feature importance analysis was done to show the relative importance of various features (symptoms in this case) in the decision tree.
